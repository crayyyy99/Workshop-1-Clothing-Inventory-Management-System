# Workshop-1-Clothing-Inventory-Management-System
Here are the codes I have written for my workshop 1 during my uni Year 2 Semester 1 (Oct 2019 ~ Jan 2020). <br/>
This system is a standalone console application created using C++.

ERD: 
![image](https://user-images.githubusercontent.com/108722803/177764520-27a72c59-5dad-4292-8039-abe64ee71efa.png)

## Scopes:
Target user: the clothes store business owners and their staffs.
* The administrators are the owner and the senior-level employees that are assigned as admin by the owner.
* The normal user is the salesperson or the cashier in the store. 

### For normal user (salesperson or cashier):
* Add, search, and view transaction.
* Search and view products
* Edit account details.

### For administrators (business owner and senior-level employees):
* Search, view, edit, and delete employee.
* Add, search, view, edit, and delete supplier.
* Add, search, view, edit, and delete product.
* Search, view, and edit transaction.
* Add, search, and view purchase orders.
* Generate sales report and best seller report.
